# Jx.Tools 聚享工具集

此工具集为收集网络上常用的一些可以在WEB运行的小工具。

## 当前工具集

1. 文本比对工具
   比对左右两个文本，查出两边不一样的行，不考虑顺序。

2. 图片转Base64工具
   将图片转换为Base64，直接在Razor里使用流转换，不在服务器存储图片文件，无泄露风险。

## 安装

该项目使用.net编写，请安装.net6后编译

### 直接编译
```
 dotnet build
```

### Docker
```
 docker build
```
