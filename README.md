# Jx.Tools 聚享工具集

此工具集为收集网络上常用的一些可以在WEB运行的小工具。

## 安装

该项目使用.net编写，请安装.net6后编译

### 直接编译
```
 dotnet build
```

### Docker
```
 docker build
```
